# AI-Powered Product Review & Sales Analytics Dashboard

## Overview

Built an end-to-end analytics solution using Python and Power BI to analyze customer reviews and product sales performance.

## Tools Used

* Python
* Pandas
* TextBlob
* Power BI
* Excel

## Key Results

* Analyzed 25+ product reviews and sales records
* Calculated average product rating of 3.97/5
* Identified highest-selling product with 35 monthly sales
* Performed sentiment analysis using NLP techniques
* Developed interactive Power BI dashboard
